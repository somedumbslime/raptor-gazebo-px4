# RAPTOR AI

RAPTOR AI — модульний проєкт наведення та супроводу цілі для БПЛА в середовищі Gazebo + PX4 SITL.

## Опис

Проєкт реалізує єдиний керований контур:

`камера -> детектор -> трекер -> вибір primary target -> guidance -> керування платформою`.

Кінцева ціль — повністю автоматичний цикл польоту:

`зліт -> пошук -> захоплення цілі -> супровід -> посадка`.

## Стан проєкту

Актуальний стан: **активна Фаза 5 (PX4/MAVLink Follow Behavior)**.

Поточна позначка на шкалі прогресу:

`[███████░░░] 7/10`

Зведений статус фаз:

- Фаза 0 — стабілізація baseline: **Завершено**
- Фаза 1 — інтеграція PTS: **Завершено**
- Фаза 2 — actor target у сцені: **Завершено**
- Фаза 3 — YOLO ONNX + tracking + PTS: **Завершено**
- Фаза 4 — перехід на дронову платформу: **Завершено**
- Фаза 5 — follow-контур на PX4/MAVLink: **В процесі**
- Фаза 6 — стабілізація та regression-gates: **Заплановано**
- Фаза 7+ — масштабування сценаріїв і системи: **Заплановано**

## Що вже реалізовано

- Runtime V2 з модульною архітектурою та стабільними контрактами між підсистемами.
- Інтеграція `primary-target-selection` через зовнішній адаптер.
- Інтеграція `target-guidance` із профілями `safe`, `balanced`, `aggressive`.
- Єдиний платформний інтерфейс для режимів `gimbal`, `iris`, `px4`.
- E2E прогони в PX4-сценах з actor-ціллю.
- Запис метрик, подій і артефактів для аналізу якості поведінки.

## Найближчі цілі

- Зменшити осциляції та підвищити стабільність follow-контролю у Фазі 5.
- Завершити формалізовану польотну state machine.
- Закрити Фазу 5 через вимірюваний closure-звіт (DoD + артефакти).
